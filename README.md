## Setup

```bash
  pip install pygame
```

Open the folder in which you wanted to run your project.

Open the terminal there and enter the following cmd

```bash
  git init
```
```bash
  git clone https://github.com/satyameshmali12/pingpong
```
    
## Controls

For player1

```bash
  W - to move up
```

```bash
  S - to move down
```

For player2

```bash
  i - to move up
```

```bash
  k - to move down
  ```
